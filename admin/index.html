Connect a Supabase project

The Watch Alley admin is not wired to a Supabase project yet.

The original project was the wrong tenant and has been removed. To finish setup:

  1. Create a new Supabase project under your own org.
  2. Run The Watch Alley migrations from docs/inventory-schema.md against it.
  3. Fill WATCH_ALLEY_SUPABASE_URL and WATCH_ALLEY_SUPABASE_ANON_KEY in .env.local.
  4. Replace the placeholder SUPABASE_URL and SUPABASE_ANON_KEY constants near the top of scripts/admin.js with the same values.
  5. Insert your email into watch_alley.admin_emails via Supabase Studio.
  6. Reload this page.
